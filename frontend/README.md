# FasoTontine Frontend

Application web Next.js, React, Tailwind CSS et PWA.

## Installation

```bash
npm install
```

## Developpement

```bash
npm run dev
```

L'application sera disponible sur http://localhost:3000.

## Structure

- `app/` : routes et layouts Next.js.
- `components/` : composants reutilisables.
- `lib/` : clients API et utilitaires.
- `public/` : manifest PWA, service worker et assets publics.
- `styles/` : styles partages optionnels.

## Carte
